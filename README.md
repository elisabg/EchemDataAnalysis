For the file to properly function, it is necessary to download Jupyter via: https://jupyter.org/install 
The jupyter notebook can be used to plot Cyclic Voltammetry and Electical Impedance Spectroscopy data from the Gamry Framework software, using Python instead of Matlab.
The downloaded .txt files need to be adjusted by deleting the XY at the beginning of running the code.
